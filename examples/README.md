# Markdown Viewer - Examples

This folder contains example files to test the rendering capabilities of Markdown Viewer.

## Files

- [Code Highlighting](code-highlighting.md) - Syntax highlighting with Highlight.js
- [Math Formulas](math-katex.md) - Mathematical expressions with KaTeX
- [Mermaid Diagrams](mermaid-diagrams.md) - Flowcharts, sequences, git graphs, pie charts
- [GitHub Alerts](alerts.md) - Note, Tip, Important, Warning, Caution blocks
- [Footnotes](footnotes.md) - References and endnotes
- [Typography & Emojis](typography.md) - Smart quotes, dashes, ellipses, emoji shortcodes
- [Extended Tables](tables-extended.md) - Colspan and complex tables
- [Task Lists](task-lists.md) - Checkbox-style task lists
- [Images](images.md) - Image rendering with lightbox zoom
- [Mixed Content](mixed-content.md) - Combining all features together
