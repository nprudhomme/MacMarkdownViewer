# Keyboard Shortcuts

## Application

| Shortcut | Action |
|---|---|
| **Cmd+O** | Open folder |
| **Cmd+Q** | Quit application |

## Overlays

| Shortcut | Action |
|---|---|
| **Escape** | Close image lightbox |
| **Escape** | Close Mermaid fullscreen view |

## Theme

The theme toggle button (sun/moon icon) in the top-right corner switches between light and dark mode. The preference is persisted across sessions.
